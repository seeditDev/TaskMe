## ✨ Features

- ✅ **Task Management** - Create, organize, and track your tasks
- 📝 **Notes** - Keep your ideas and thoughts in one place
- 🏷️ **Tags & Folders** - Organize with custom tags and folders
- 🔔 **Reminders** - Get notified before due dates
- 🔊 **Voice Feedback** - Audio confirmations for actions
- 📊 **History** - View all your activities
- 🔄 **Auto-Update** - Automatic app updates from GitHub
- 📴 **Offline First** - Works without internet
- 🔒 **Privacy** - Your data stays on your device

## 🚀 Installation

### Android (APK)

1. Download the latest APK from [Releases](https://github.com/seeditDev/TaskMe/releases/latest)
2. Open the APK file on your Android device
3. If prompted, allow "Install from unknown sources"
4. Complete installation and open TaskMe

### System Requirements
- Android 5.0 (API 21) or higher
- ~20 MB storage space

---

## 🔄 Auto-Update

TaskMe includes an automatic update system:

1. App checks GitHub for updates on launch (once per day)
2. If update available, shows notification with changelog
3. Tap "Update Now" to download and install automatically
4. App restarts with new version

### Manual Check
Go to **More** → **Check for Updates**

## 🛠️ Building from Source

### Prerequisites
- Node.js 18+
- Android Studio (for Android build)
- Java JDK 17

### Setup

```bash
# Clone repository
git clone https://github.com/seeditDev/TaskMe.git
cd TaskMe

# Install dependencies
npm install

# Start development server
npx expo start

# Build Android APK
cd android
./gradlew assembleRelease
```
