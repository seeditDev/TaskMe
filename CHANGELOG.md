# Changelog

All notable changes to TaskMe will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.2] - 2024-01-15

### Added
- 🔄 GitHub auto-update system
- 📱 APK download and install flow
- 📊 Activity history tracking
- 🔊 Voice feedback for actions
- 🏷️ Enhanced tag management
- 🔔 Pre-due reminders
- 🌐 Website integration (seedit.site/taskme)
- ✨ QR code sharing support

### Changed
- Settings tab renamed to "More"
- Improved UI with status-based task containers
- Enhanced notification handling

### Fixed
- Reminder notification sync issues
- Task status update from notifications
- UI refresh after notification actions

---

## [1.0.1] - 2024-01-01

### Added
- 🔔 Push notifications for task reminders
- 📱 Android notification channels
- 🎨 Improved color scheme
- 📊 Dashboard statistics
- 🔄 Data backup and restore
- 📤 Data export/import functionality

### Changed
- Enhanced task due date handling
- Improved storage management

### Fixed
- Date picker issues on Android
- Notification scheduling bugs

---

## [1.0.0] - 2023-12-15

### Added
- 🎉 Initial release of TaskMe
- ✅ Task management (CRUD operations)
- 📝 Notes with rich text support
- 🏷️ Tags and folders organization
- 🔔 Due date reminders
- 📱 Cross-platform mobile app
- 🔒 Local SQLite database
- 🎨 Modern, clean UI
- 🌙 Dark mode support
